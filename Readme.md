##### Pong Game ####

Created by Wei Fang

For Game Engineering 3D course

Using Ogre game display engine

Operation:

Single Player:

1P: W S A D OR ↑ ↓ ← →

Two Player:

1P: W S A D

2P: ↑ ↓ ← →


Source code and demo included

Screenshots included in Screenshot.docx
